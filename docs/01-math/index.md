# Part 1 · 数学基础

深度学习的本质是在高维空间里做优化。读懂论文里的公式，需要三块数学：线性代数（描述数据和变换）、概率论（描述不确定性）、优化理论（描述如何训练）。

本章不追求数学严谨，目标是让你"用得上"。
