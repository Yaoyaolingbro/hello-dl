# 线性代数

矩阵是深度学习的基本语言：数据是矩阵，权重是矩阵，注意力分数也是矩阵。这一章从基础工具出发，逐层递进到科研中真正会用到的高级技巧。

## 本章知识地图

```mermaid
graph LR
    A[向量与范数] --> B[矩阵运算与性质]
    B --> C[特殊矩阵]
    C --> D[矩阵求导]
    D --> E[链式法则]
    C --> F[特征值分解]
    F --> G[SVD]
    G --> H[复数与Euler公式]
    C --> I[叉积与刚体运动]
    H --> J[几何变换与相机模型]
    I --> J
    F --> K[图的矩阵表示]
```
