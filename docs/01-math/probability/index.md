# 概率论

生成模型、损失函数、贝叶斯方法——这些都是概率论的语言。这一章打通概率基础和深度学习的连接。

```mermaid
    E --> F[信息论\n熵/KL散度]
    F --> G[变分推断\nELBO]

    C --> H

    I --> K
    J --> K
```
