# 深度学习入门指南

欢迎。这本书写给刚开始学 AI 的同学——有大学数学基础，想系统搞懂深度学习，能看懂论文，也能应对面试。

## 如何使用这本书

不一定要按顺序读完 Part 1（数学基础）再看后面的部分，但希望大家能有好的数学基础。

如果你对某个概念不熟悉，可以先跳过去看后面章节的直觉段，看看它在解决什么问题，再回过头来读数学部分。
