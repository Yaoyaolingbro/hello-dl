# 深度学习入门指南

欢迎。这本书写给刚开始学 AI 的同学——有大学数学基础，想系统搞懂深度学习，能看懂论文，也能应对面试。

## 如何使用这本书

按顺序读完 Part 1（数学基础）再进入 Part 2，是最稳的路径。
如果你数学基础不错，可以直接从 Part 2 开始，遇到不熟悉的概念再回来查。
