# Part 1 数学基础写作规范

> 本文件与 `writing-style.md` 配合使用，**不重复通用规则，只写 Part 1 特有的要求**。

---

## 核心定位

读者有大学数学基底，但没有在科研里用过。目标是：**读完能看懂论文里的公式，而不是通过数学考试**。

严谨和易读不是对立的。严谨意味着每个符号有定义、每步推导有依据；易读意味着在符号之前先说清楚"这在干什么"。

---

## 章节结构

每个数学小节的固定结构：

```
## 直觉 (Intuition)          ← 最多 5 句，必须放第一位；可在此处插入示意图

## 定义与性质                ← 先说结论，再给出形式化定义
                               ← 每个关键性质后跟一个 2D/3D 数值例子

## 推导 / 证明思路           ← 分步写，每步说目的（可选，视难度决定深度）

## 在深度学习中的应用        ← 明确说"这个数学工具出现在哪里"
```

**不写代码**：Part 1 数学基础章节用手算数值例子替代代码验证，用示意图替代可视化输出。

---

## 定义的写法

先给出直觉描述，再给出形式化定义，不要反过来。

**不好的写法：**
> 设 $\mathbf{A} \in \mathbb{R}^{n \times n}$ 为实对称矩阵，若对所有非零向量 $\mathbf{x}$ 有 $\mathbf{x}^\top \mathbf{A} \mathbf{x} > 0$，则称 $\mathbf{A}$ 为正定矩阵。

**好的写法：**
> 正定矩阵 (Positive Definite Matrix) 可以理解为"对任何方向都有拉伸效果，没有压缩或翻转"的矩阵。形式上，若对所有非零向量 $\mathbf{x}$ 都满足：
> $$\mathbf{x}^\top \mathbf{A} \mathbf{x} > 0$$
> 就称 $\mathbf{A}$ 正定。

---

## 定理的写法

重要结论用 `!!! note "定理名"` 框出，然后给出两种层次的解释：

```markdown
!!! note "谱定理 (Spectral Theorem)"
    实对称矩阵一定可以正交对角化：$\mathbf{A} = \mathbf{Q} \mathbf{\Lambda} \mathbf{Q}^\top$，
    其中 $\mathbf{Q}$ 的列是单位正交特征向量，$\mathbf{\Lambda}$ 是特征值对角矩阵。

直觉：对称矩阵在自己的"自然坐标系"里就是纯粹的缩放操作。
```

## 推导深度的判断标准

| 情形 | 处理方式 |
|------|----------|
| 结论反直觉，读者容易记错 | 写完整推导，每步说目的 |
| 结论直觉上显然，推导是机械计算 | 给出关键步骤，其余"按类似方法展开" |
| 结论来自高阶分析（如泛函分析） | 只给出结论 + 直觉解释 + 参考文献 |

不要"为了严谨"把所有定理都完整证明——这是教科书干的事，不是入门教程干的事。

## 示意图规范（数学章节）

**每小节至少插入一张示意图**，没有示意图的章节是不完整的。图的来源和嵌入规范见 `writing-style.md` 的"图的使用规范"一节。

数学章节常用图的类型及搜索词：

| 概念 | 推荐搜索词 |
|------|-----------|
| 向量、内积、投影 | `dot product projection visualization` |
| L1/L2 范数单位球 | `L1 L2 norm unit ball comparison` |
| 特征向量变换 | `eigenvector matrix transformation` |
| SVD 几何含义 | `SVD singular value decomposition geometry` |
| 梯度下降 | `gradient descent loss landscape` |
| 正定矩阵二次型 | `positive definite quadratic form bowl` |
| 正交投影 | `orthogonal projection vector subspace` |

嵌入格式（Caption 必须说明图展示了什么）：

```markdown
![向量内积的几何含义](图片URL)

*两向量夹角越小，内积越大；正交时内积为零。来源：[Wikipedia](链接)*
```

## 数值例子规范

不写代码，改用手算数值例子：

- 使用 2D 或 3D 具体数字（避免 $n$ 维泛化）
- 给出完整计算过程，最后写验证等式（如 $=\mathbf{A}\checkmark$）
- 几何类概念给出几何解释（"方向不变"、"面积缩放 $k$ 倍"等）

## Admonition 使用

- `!!! note "定理名"` — 包裹重要结论
- `!!! tip "在深度学习中的应用"` — 点明这个数学工具被用在哪里，1-2 句即可
- `!!! warning "常见符号混淆"` — 容易写错的符号或下标
- **不用** `!!! tip "面试重点"` — 数学章节不面向面试
